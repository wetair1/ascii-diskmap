# ascii-diskmap

ASCII curses disk usage mapper for directories.

## Features

- Scans files and folders
- Sorts entries by size
- Draws ASCII usage bars
- Scroll with keyboard
- Pure Python stdlib

## Usage

```bash
python3 main.py
python3 main.py /path/to/folder
```

## Controls

- `j` / Down: scroll down
- `k` / Up: scroll up
- `q`: quit
